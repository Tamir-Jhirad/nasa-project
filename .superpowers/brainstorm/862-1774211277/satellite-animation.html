Satellite Orbit Visualization Style

When you select a satellite on the 3D globe, what should you see?

B — Static Orbit Ring

Full orbital path drawn as a ring around Earth. Same style as the asteroid orbits. Simple, always accurate, no animation overhead.

  • Instant, no math per frame
  • Works for all ~8k sats
  • Doesn't show current position

A — Real-time Live Position

A glowing dot moves along the orbit at the satellite's actual speed. Requires satellite.js to propagate TLE every second.

  • Shows where sat is RIGHT NOW
  • Satisfying to watch
  • Only practical when 1-few selected

C — Ring + Live Position Dot

Static orbit ring (always visible) plus an animated dot for current position. Best of both — clear orbital path and real location.

  • Most informative
  • Ring = context, dot = now
  • Slightly more complex
